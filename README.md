# MediCard 医杀 — 医学卡牌对战游戏

> 三国杀机制 × 医学题库：通过卡牌对战高效复习基础医学知识，支持多人实时联机。

## 游戏简介

**MediCard 医杀** 是一款基于浏览器的医学知识卡牌对战游戏。核心玩法借鉴三国杀的回合制卡牌机制，但将"杀/闪/桃"替换为医学主题卡牌，**所有卡牌效果通过答题判定**——答对生效，答错作废。

- 🃏 **120+ 张卡牌**：基础牌（攻击/防御/治疗）、锦囊牌、装备牌、延时锦囊
- 📚 **8 个医学学科**：细胞生物学、生物化学、生理学、病理学、组胚、解剖学、免疫学、微生物学
- 👥 **2~5 人身份局**：主公、忠臣、反贼、内奸，各有专属技能
- 🤖 **单人练习模式**：AI 对手，三档难度可选
- 🌐 **P2P 联机对战**：基于 WebRTC，房主创建房间，输入 6 位房间号加入
- 📱 **手机优先设计**：响应式布局，完美兼容桌面端

## 身份系统

| 身份 | 血量 | 技能 |
|------|------|------|
| 主公 👑 | 5❤️ | 初始血量+1 |
| 忠臣 🛡️ | 4❤️ | 可替主公挡刀 |
| 反贼 💀 | 4❤️ | 攻击主公时伤害+1 |
| 内奸 🕵️ | 4❤️ | 每回合多摸1张牌 |

## 技术栈

- **前端**：HTML5 + CSS3 + Vanilla JavaScript（模块化架构）
- **联机**：PeerJS (WebRTC P2P)
- **构建**：Node.js 构建脚本，开发时模块化 → 构建合并为单文件
- **部署**：纯静态文件，可部署到 Cloudflare Pages / GitHub Pages / 任意静态服务器
- **加速**：内置 Speed Engine v2（HTML预构建缓存 + 事件委托 + Web Worker导出）

## 快速开始

### 开发模式

```bash
# 安装依赖（仅构建工具需要）
npm install

# 直接用浏览器打开 index.html 即可游玩
# 或使用任意静态服务器
npx serve .
```

### 构建生产版本

```bash
npm run build
# 输出到 dist/index.html
```

## ☁️ Cloudflare Pages 部署（推荐）

### 一键部署（2步）

**第 1 步**：在 [Cloudflare Pages](https://dash.cloudflare.com/pages) 后台点击「连接到 Git」，选择此仓库。

**第 2 步**：配置全部使用默认值，点击「保存并部署」：
- **构建命令**：留空（纯静态站点无需构建）
- **输出目录**：`/`（根目录）
- **框架预设**：无

部署完成后，网站地址为：`https://你的项目名.pages.dev`

### 后续更新

修改代码并推送到 GitHub `main` 分支，Cloudflare Pages 自动重新部署。

```bash
git add . && git commit -m "描述你的修改" && git push
```

### 重要说明

- ✅ **单人练习、刷题工坊、错题本**：完全可用（本地存储）
- ✅ **主题切换、加速引擎**：完全可用
- ⚠️ **多人联机**：需要 PeerJS 信令服务器（已配置免费中继）
- ⚠️ **账号同步**：API 调用会静默失败，不影响核心功能

## 项目结构

```

├── index.html              # 开发入口

│   │   ├── audio-system/   # 音频系统
│   │   └── storage/        # 本地存储
│   ├── css/                # 样式（按模块拆分）
│   ├── lib/                # 外部库
│   └── assets/             # 素材
├── server/                 # 信令服务器
├── tools/                  # 构建/测试工具
└── design.txt              # 完整设计文档
```
